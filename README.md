# Portofolio GitHub

Template portofolio sederhana dengan:

- `index.html`
- `styles.css`
- `script.js`

## Cara pakai

1. Buka `index.html` di browser.
2. Edit nama, deskripsi, proyek, dan link kontak.
3. Upload file ke repository GitHub.

## Deploy gratis

Kamu bisa tampilkan portofolio ini online dengan GitHub Pages:

1. Push project ke GitHub.
2. Masuk ke repository.
3. Buka **Settings** > **Pages**.
4. Pada **Build and deployment**, pilih branch utama dan folder `/root`.
5. Simpan, lalu tunggu link website muncul.
